## Description
This is a Netlify plugin for caching files. 

## Installation
To install, run the following command:
```
npm install cache-netlify-plugin
```

## Usage
To use the plugin, add the following to your `netlify.toml` file:
```
[[plugins]]
  package = "cache-netlify-plugin"
```
